# Rumi-APT
___
## Infomation
Rumi-APT is runs on the Windows and LINUX.<BR>
Rumi-APT mean is APT in LINUX.<BR>
[More Infomation](https://rumiserver.com/rumistore/RumiAPT.html)

## Use
### LINUX↓
```sh
/home/User$ rapt help
```
### Windows↓
```sh
C:\Users\User> rapt help
```
or
```sh
C:\Users\User> apt help
```
